# 项目指令（叠加模板）

仅当兼容的全局指令文件已经生效时使用本模板。它只补充已验证的项目事实和项目级约束，不替代、不削弱全局安全、敏感信息、删除或确认边界。

只有项目中已验证的信息或用户明确提供的信息才能替换字段。未知信息应留空、标记为待确认，或只提出最少的必要问题。不得猜测命令、路径、目录职责、部署流程或 Agent 配置位置。
