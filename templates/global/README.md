# Global Templates

Choose [AGENTS.md](AGENTS.md) for an English default or [AGENTS.zh-CN.md](AGENTS.zh-CN.md) for a Simplified Chinese default. These templates define stable cross-project collaboration preferences and safety boundaries; they do not contain project commands, directory responsibilities, technology-stack claims, or installation paths.

The two files have the same four navigation groups and 23 ordered rule units. Only the default user-facing language, technical-term expression, and default Git commit-message language are localized. See the [template index](../README.md#global-rule-mapping-and-localization-review) for the semantic mapping record.

Before installing a global template, discover the actual Agent configuration and any existing rules. Prepare a reviewable candidate, show the target, rationale, diff, verification, and rollback, and wait for explicit confirmation before writing. The stage 4 installation guide is not available yet.
