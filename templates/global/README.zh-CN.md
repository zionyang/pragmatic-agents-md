# 全局模板

如需英文默认版本，选择[AGENTS.md](AGENTS.md)；如需简体中文默认版本，选择[AGENTS.zh-CN.md](AGENTS.zh-CN.md)。这些模板定义跨项目稳定生效的协作偏好和安全边界；其中不包含项目命令、目录职责、技术栈断言或安装路径。

两份文件具有相同的四个导航分组和 23 条有序规则单元。只有默认用户可见语言、技术术语表达和默认 Git 提交信息语言会因本地化而不同。语义映射记录见[模板索引](../README.zh-CN.md#global-rule-mapping-and-localization-review)。

安装全局模板前，先发现实际 Agent 配置和既有规则。准备可审阅候选，展示目标、理由、diff、验证与回滚方式，并在写入前等待明确确认。详见[安装指南](../../guides/INSTALL.zh-CN.md)、[裁剪指南](../../guides/CUSTOMIZE.zh-CN.md)和[规则说明](../../GUIDE.zh-CN.md)。这些指南均不为未知 Agent 确定配置路径，也不承诺自动加载。
